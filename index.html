Nossa Missão

Automatizando processos e simplificando o desenvolvimento

A squad Low Code Hub é responsável por automatizar processos, facilitar o trabalho dos desenvolvedores e cuidar das principais plataformas de Low Code da organização.

Plataformas

Power Automate

Crie fluxos automatizados para sincronizar arquivos, receber notificações e coletar dados.

Automation Anywhere

Automação robótica de processos (RPA) para tarefas repetitivas.

ServiceNow

Plataforma de gestão de serviços para automação de fluxos de trabalho de TI.

Sobre a Squad

A squad é especializada em diminuir a carga operacional dos desenvolvedores, automatizando processos repetitivos e fornecendo ferramentas que facilitam o desenvolvimento de aplicações.
